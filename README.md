

# Virtual Hospital Management System

This is a database course project for a virtual hospital management system.

## Technologies
- MySQL
- Java
- Java Swing
- JDBC

## Main Features
- Manage patients, doctors, appointments, bills, and payments
- Dashboard counters
- Reports using SQL JOIN queries
- Database schema, DML operations, and database dump included

## How to Run
1. Restore the database dump.
2. Open the Java project in IntelliJ.
3. Add MySQL Connector/J.
4. Update DBConnection.java with your database information.
5. Run HospitalApp.java.
